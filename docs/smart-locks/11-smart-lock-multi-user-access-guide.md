# Smart Lock Multi-User Access Guide

Guide to managing multiple users on UK smart locks covering user types, permission levels, activity tracking, temporary codes, and best practices for household members, cleaners, and guests.

---

## User Types and Permission Levels

Smart locks support different user types: Admin users have full control (add/remove users, change settings, view activity log) - limit to homeowners. Standard users can lock/unlock and view their own activity. Guest users have restricted access, typically limited to specific time windows or single-use codes. Most apps support 20-50 users total. The Yale Home app supports 20 users, Nuki supports 100, Ultion supports 50. Each user gets a unique PIN code (4-8 digits) and optional app access. Activity is logged per-person, creating an audit trail showing who entered and when.

---

## Managing Users Effectively

Create individual codes for each user - never share one PIN. Name each user clearly for the activity log. Use temporary codes for short-term access: time-limited (1 hour to 30 days), single-use (valid for one unlock), or recurring schedules (cleaner Tuesday mornings). Revoke access immediately when no longer needed. Review the user list monthly and remove inactive users. For Airbnb, each guest gets a unique temporary PIN valid only during their stay. Set auto-lock to 30 seconds. Enable notifications for unusual activity - multiple failed attempts, unlocks at unusual hours, or door left open.

---

## Specifications and Comparison

| User Type | Access Level | Typical Use | Recommended Count |
| ---|---|---|--- |
| Admin | Full control, manage users | Homeowner | 1-2 per household |
| Standard | Lock/unlock, view own activity | Family members | 2-6 per household |
| Guest (time-limited) | Lock/unlock during set window | Cleaners, contractors | As needed |
| Guest (single-use) | One-time unlock | Delivery drivers | Rarely used |
| Airbnb guest | Time-bound, no app access | Short-term renters | Per booking |

## FAQ

<dl>

<dt>How many users can I add?</dt>
<dd>Most smart locks support 20-50 users. Nuki supports up to 100. Each user gets a unique PIN and optional app access.</dd>

<dt>Can I see who used the lock and when?</dt>
<dd>Yes - the activity log shows which user code or app account was used with timestamps.</dd>

<dt>How do I remove a users access?</dt>
<dd>Delete their user profile in the app. This immediately revokes their PIN code and app access. For more information see our <a href="../smart-locks/10-smart-lock-app-setup-and-configuration.md">Smart Lock App Setup and Configuration</a>.</dd>

</dl>
