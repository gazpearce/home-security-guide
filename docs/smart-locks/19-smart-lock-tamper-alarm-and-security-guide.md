# Smart Lock Tamper Alarm and Security Guide

Guide to smart lock tamper alarm features for UK homes covering tamper detection types, alarm configuration, push notifications, and response to attempted break-ins.

---

## Tamper Detection Technology

Smart locks include multiple tamper detection systems. Forced entry detection: accelerometer detects impact (hammer, crowbar, drill). Triggers internal alarm and push notification. Removal detection: tamper switch triggers if keypad is forcibly removed. Multiple failed code alert: after 5-10 incorrect attempts, lock enters lockout (30s-5min) and sends alert. Door ajar alarm: detects if door left open beyond set time. Low-battery warning: not a tamper alert but critical for security. Popular UK models: Yale Conexis L1 (100dB internal alarm, push notifications), Nuki Smart Lock 3.0 (accelerometer, forced entry detection), Ultion Smart (three sensitivity levels), Lockly Secure Pro (built-in alarm).

---

## Configuring Tamper Alarms and Responses

Step 1: Enable all tamper alarms in lock settings. Step 2: Set failed code threshold to 5 attempts with 1-minute lockout. Step 3: Set forced entry sensitivity to medium - high may false alarm from slamming doors, low may miss attacks. Step 4: Set door ajar timer to 2 minutes. Step 5: Enable push and email notifications for all events. Step 6: Test after installation - gently knock the lock to verify vibration sensor activates. Step 7: If tamper alarm triggers when away, check camera footage before returning. Step 8: Coordinate with overall security system - Ring alarm can trigger sirens when lock tamper detected.

## Specifications and Comparison

| Tamper Type | Detection Method | Alarm Action | Notification Type |
| ---|---|---|--- |
| Forced entry | Accelerometer/vibration sensor | Internal siren 100dB | Push + email |
| Removal | Tamper switch on keypad | Immediate siren | Push + email |
| Failed codes | PIN attempt counter | Lockout + alert | Push notification |
| Door ajar | Magnetic contact | Alert after timer | Push notification |
| Low battery | Voltage monitoring | Intermittent beep | Push + in-app |

## FAQ

<dl>

<dt>Can a tamper alarm alert the police?</dt>
<dd>No - it sends notifications to your phone. For monitored response, integrate with a Grade 2 alarm system with ARC monitoring.</dd>

<dt>Will the tamper alarm drain the battery?</dt>
<dd>Negligible impact - sensors use microamps in standby. The siren uses more power but sounds for only 30-60 seconds.</dd>

<dt>Can I silence a false tamper alarm?</dt>
<dd>Yes - acknowledge the alert in the app. Adjust sensitivity if false alarms occur frequently. For more information see our <a href="../smart-locks/18-smart-lock-emergency-override-guide.md">Smart Lock Emergency Override Guide</a>.</dd>

</dl>
