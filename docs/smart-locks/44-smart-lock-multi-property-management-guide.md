## Specifications and Comparison

| Portfolio Size | Recommended Platform | Properties Supported | Cost | Key Feature |
| ---|---|---|---|--- |
| 2-4 properties | Nuki or Yale app | 10-20 homes | Free | Simple multi-home profiles |
| 5-20 properties | Ultion Pro | 50+ properties | Free (under 10) / subscription | Dashboard, bat monitoring |
| 20+ properties | PMS integration (IZY) | 500+ properties | Subscription | Auto code generation |
| HMO portfolio | Ultion Pro + fire log | Unlimited | Subscription | Fire log integration |
| Holiday lets (Airbnb) | IZY / Igloohome | 100+ properties | Per-property fee | Booking platform sync |

## FAQ

<dl>

<dt>Can I manage locks for multiple properties from one app?</dt>
<dd>Yes - Nuki and Yale apps support multiple property profiles. Ultion Pro provides a dedicated dashboard for portfolio management.</dd>

<dt>How do I manage codes for multiple holiday let properties?</dt>
<dd>Use platform integration (IZY, Igloohome Connect) to auto-generate guest codes from Airbnb/Booking.com bookings. Codes auto-expire at check-out.</dd>

<dt>What is the most important feature for multi-property management?</dt>
<dd>Centralised battery monitoring - dead batteries cause lockouts across remote properties. Check battery status weekly from a single dashboard. For more information see our <a href="../smart-locks/43-smart-lock-anonymous-entry-and-privacy-mode-guide.md">Smart Lock Anonymous Entry and Privacy Mode Guide</a>.</dd>

</dl>
