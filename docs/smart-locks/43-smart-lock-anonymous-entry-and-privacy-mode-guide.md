## Specifications and Comparison

| Privacy Feature | What It Does | When to Use | Available On |
| ---|---|---|--- |
| Privacy Lock | Prevents external unlock | Night-time, home alone | Nuki, Yale, Ultion |
| Anonymous PIN code | No user ID in activity log | Contractors, emergency access | Lockly, Ultion |
| Privacy schedule | Auto-enable privacy 10pm-7am | Nightly | Nuki (via schedule) |
| Guest privacy | Suppress guest notifications | Holiday let guests | Nuki, Yale, Ultion |

## FAQ

<dl>

<dt>What happens when privacy mode is on?</dt>
<dd>The lock cannot be unlocked from outside via keypad or app. The interior thumb-turn still opens the door. Privacy mode is designed for when you are home.</dd>

<dt>Can I create codes that do not show in the activity log?</dt>
<dd>Yes - anonymous or privacy codes are available on Lockly and Ultion. These codes grant access but do not log the user identity.</dd>

<dt>Will privacy mode affect holiday let operations?</dt>
<dd>Configure guest privacy - suppress notifications during guest stay. Keep the activity log for security (timestamps only) but suppress user identification. For more information see our <a href="../smart-locks/42-smart-lock-temperature-and-humidity-effects-guide.md">Smart Lock Temperature and Humidity Effects Guide</a>.</dd>

</dl>
