## Specifications and Comparison

| Metal Door Type | Thickness | Cylinder Size | Smart Lock Suitability | Installation Note |
| ---|---|---|---|--- |
| Steel fire door (flat/HMO) | 44-54mm | 40/40mm or 45/45mm | Yale Conexis L1, Nuki | Use adhesive mount, avoid drilling |
| Metal-clad composite | 44-52mm | 35/45mm typical | Ultion Smart, Yale | Dielectric grease on threads |
| Industrial steel door | 50-80mm | Non-standard possible | Nuki Pro, specialist | Consult manufacturer |
| Security-rated door (LPS) | 50-80mm | Reinforced cylinder | Specialist only | Check LPS compliance |
| Steel framed glass door | 40-50mm | Standard euro | All major brands | Insulate electronics from frame |

## FAQ

<dl>

<dt>Can I install a smart lock on a fire door?</dt>
<dd>Yes - cylinder replacement is acceptable. Do not drill through the steel skin. Use adhesive mounting for external components. Check with the building fire risk assessor.</dd>

<dt>Will a metal door affect wireless signal?</dt>
<dd>Yes - steel doors cause 20-40% wireless signal reduction. Position WiFi router within 8m of the door. Consider a Bluetooth range extender.</dd>

<dt>Do I need special tools for metal door installation?</dt>
<dd>Standard tools plus: metal drill bits if drilling is needed (avoid if possible), dielectric grease for corrosion prevention, and rubber gaskets for electrical isolation. For more information see our <a href="../smart-locks/37-smart-lock-energy-efficiency-and-standby-power-guide.md">Smart Lock Energy Efficiency and Standby Power Guide</a>.</dd>

</dl>
