## FAQ

<dl>

<dt>Can a smart lock be used in an HMO property?</dt>
<dd>Yes - smart locks are suitable for HMOs. Ensure the lock does not compromise fire door integrity and allows escape without key. Check with your fire risk assessor.</dd>

<dt>How do I prevent tenant lockout in a rental?</dt>
<dd>Provide a key safe with physical key backup. Share emergency code with managing agent. Set low-battery alerts to the landlord app as well as tenant app.</dd>

<dt>Can smart locks help with deposit disputes?</dt>
<dd>Yes - the activity log provides evidence of entry/exit times which can be used in tenancy deposit disputes. Keep logs for the duration of the tenancy. For more information see our <a href="../smart-locks/24-smart-lock-encryption-and-data-security-guide.md">Smart Lock Encryption and Data Security Guide</a>.</dd>

</dl>
