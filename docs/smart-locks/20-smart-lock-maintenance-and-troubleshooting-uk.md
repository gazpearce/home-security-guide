# Smart Lock Maintenance and Troubleshooting UK

Complete guide to smart lock maintenance and troubleshooting for UK homes covering routine care, common problems, connectivity fixes, door alignment corrections, and when to call a locksmith.

---

## Routine Smart Lock Maintenance

Monthly: clean keypad with damp cloth, check for wear, test emergency override methods, review activity log. Quarterly: check door alignment (drooping door drains batteries), lubricate cylinder with graphite powder (not oil), check mounting screws, test auto-lock 5 times. Annual: replace batteries proactively (even if app shows good level), remove lock and clean cylinder hole, check weather seals, update firmware, tighten all fixings. Graphite lubricant every 6 months - never use WD-40 which attracts dirt.

---

## Common Problems and Solutions

Problem 1: Lock wont connect to WiFi - check within 15m of router, ensure 2.4GHz band, restart lock (remove batteries 30s), restart router. UK stone walls may need WiFi extender. Problem 2: Stiff operation or grinding - check door alignment, adjust hinges, lubricate cylinder with graphite. If persists, contact manufacturer. Problem 3: Auto-lock not working - check door position sensor alignment (within 5mm), clean surfaces, recalibrate via app. Problem 4: Keypad unresponsive - check/replace batteries, clean surface, check for damage. Problem 5: False tamper alarms - reduce sensitivity, check secure mounting. When to call a locksmith: mechanism jammed, cylinder seized, door needs hinge adjustment beyond basic tuning, or professional removal needed.

## Specifications and Comparison

| Problem | Likely Cause | Quick Fix | When to Call Pro |
| ---|---|---|--- |
| No WiFi connection | Distance or 2.4GHz band | Move router closer, check 2.4GHz | If extender does not help |
| Stiff operation | Door misalignment | Adjust hinges, lubricate cylinder | If grinding persists |
| Auto-lock fails | Sensor misalignment | Recalibrate, clean sensor | If recalibration fails |
| Keypad unresponsive | Low battery or dirt | Replace batteries, clean | If new batteries do not help |
| False tamper alarms | High sensitivity or vibration | Reduce sensitivity | If frequent after adjustment |

## FAQ

<dl>

<dt>How often should I maintain my smart lock?</dt>
<dd>Clean keypad monthly, check alignment quarterly, replace batteries annually. Graphite lubricant every 6 months.</dd>

<dt>Can I use WD-40 on my smart lock?</dt>
<dd>No - WD-40 is a solvent, not a lubricant. Use graphite powder for the cylinder. WD-40 attracts dirt and damages electronics.</dd>

<dt>What causes battery drain in winter?</dt>
<dd>Cold UK temperatures reduce capacity by 20-30%. Lock motor works harder as door materials contract. Use lithium batteries for cold conditions. For more information see our <a href="../smart-locks/19-smart-lock-tamper-alarm-and-security-guide.md">Smart Lock Tamper Alarm and Security Guide</a>.</dd>
