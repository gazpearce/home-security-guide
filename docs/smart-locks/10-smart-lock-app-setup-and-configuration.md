# Smart Lock App Setup and Configuration

Guide to smart lock app setup and configuration for UK homes covering account creation, lock pairing, user management, notification settings, and advanced configuration options.

---

## Choosing and Installing the Right Smart Lock App

Each smart lock brand uses its own app. Major UK apps include Yale Home, Nuki App, Ultion App, and Lockly App. The app is your primary interface for managing access, configuring settings, and monitoring lock activity. Installation: download from the Apple App Store or Google Play Store. Create an account with a strong password. The app guides you through Bluetooth/WiFi pairing. Grant location permissions for geofencing auto-unlock and notification permissions for lock activity alerts. Most apps offer guest management for sharing access with family, friends, or service providers.

---

## Configuration Settings and Best Practices

Essential settings: Auto-Lock Timer - set to 30 seconds after door closes. Lock Sound - enable audible confirmation unless near bedrooms. Notifications - enable push for all lock/unlock events and low-battery warnings. User Codes - create individual PIN codes per person for activity tracking. Privacy Mode - disables notifications during set hours. Vacation Mode - restricts all codes except master when away. Firmware Updates - enable automatic updates for security patches. Review the activity log periodically - most apps show the last 30-90 days. This audit trail is valuable for security monitoring and for Airbnb hosts managing guest access.

---

## Specifications and Comparison

| App Feature | Purpose | Recommended Setting |
| ---|---|--- |
| Auto-Lock Timer | Auto-locks after door closes | 30 seconds |
| Notifications | Lock/unlock alerts | Enable all (push + app) |
| User Codes | Individual PIN per person | One code per household member |
| Activity Log | Audit trail of lock events | Review monthly |
| Firmware Updates | Security patches + features | Enable automatic updates |

## FAQ

<dl>

<dt>Can I control my smart lock without the app?</dt>
<dd>Yes - all smart locks include a physical key override. Keypad models work by entering a PIN code without the app.</dd>

<dt>Do I need internet for the app to work?</dt>
<dd>Bluetooth control works without internet. Remote access, notifications, and guest management require internet.</dd>

<dt>Can multiple family members use the same app?</dt>
<dd>Yes - create individual user accounts. Each person logs in with their own credentials for personalised settings. For more information see our <a href="../smart-locks/09-smart-lock-battery-replacement-guide.md">Smart Lock Battery Replacement Guide</a>.</dd>

</dl>
