# Smart Lock Security Audit Guide

Guide to auditing smart lock security for UK homes covering physical security checks, digital security assessment, firmware verification, and insurance compliance review.

---

## Physical Security Audit

Check 1: Cylinder rating - confirm TS 007 3-star or Sold Secure Gold. Check 2: Installation quality - secure cylinder, firm keypad, properly attached motor unit. Check 3: Door and frame - door fits snugly, strike plate with 75mm screws. Check 4: Physical key override - test operation, store emergency key in TS 007 key safe. Check 5: Tamper resistance - external keypad should have tamper protection. Test the tamper alarm. For a comprehensive audit, check all five areas quarterly.

---

## Digital Security Assessment

Check 1: Encryption - verify AES-128 or AES-256. Check 2: App security - strong password (12+ chars) and biometric login. Check 3: Two-factor authentication - enable on app account. Check 4: Firmware - running latest version. Check 5: Account audit - review user list, remove inactive accounts. Check 6: Activity log - review for suspicious activity. Check 7: Notifications - verify all security notifications enabled. Check 8: Insurance compliance - confirm setup meets requirements. Most UK insurers accept TS 007 3-star rated smart locks. Document findings for insurance records.

---

## Specifications and Comparison

| Audit Area | Check Item | Frequency | Action if Failed |
| ---|---|---|--- |
| Cylinder Rating | TS 007 3-star | Annually | Replace cylinder if below 3-star |
| Installation | Secure mounting, no wobble | Every 6 months | Retighten or reinstall |
| Firmware | Latest version | Monthly | Update via app |
| App Security | Strong password + 2FA | Every 3 months | Change password, enable 2FA |
| User Audit | Remove unused accounts | Monthly | Delete inactive users |

## FAQ

<dl>

<dt>How often should I audit my smart lock?</dt>
<dd>Basic check monthly (firmware, user list, activity log). Full physical inspection every 6 months.</dd>

<dt>What is the most important security feature?</dt>
<dd>The cylinder rating - a TS 007 3-star cylinder resists physical attack. The best digital security is useless if the lock can be snapped.</dd>

<dt>Should I tell my insurer I have a smart lock?</dt>
<dd>Yes - most UK insurers offer discounts for TS 007 3-star rated locks. Failure to declare may invalidate your policy. For more information see our <a href="../smart-locks/14-smart-lock-alexa-google-home-integration-guide.md">Smart Lock Alexa Google Home Integration Guide</a>.</dd>
