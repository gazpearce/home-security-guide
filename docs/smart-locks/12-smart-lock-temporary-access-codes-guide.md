# Smart Lock Temporary Access Codes Guide

Guide to creating and managing temporary access codes for UK smart locks covering time-limited codes, single-use codes, recurring schedules, and best practices for guests and service providers.

---

## Temporary Code Types and Use Cases

Temporary access codes allow you to grant access to specific people for specific periods without sharing your permanent code. Three types: time-limited codes (valid within a defined window e.g. cleaner 9-11am Tuesdays), single-use codes (expire after first use e.g. delivery driver), and recurring schedule codes (repeat weekly e.g. dog walker 1-3pm weekdays). Yale Home supports codes from 1 hour to 365 days, Nuki from 1 hour to unlimited, Ultion from 1 hour to 90 days. Time-limited codes are ideal for Airbnb guests, single-use for emergency access, and recurring for regular service providers.

---

## Creating and Managing Temporary Codes

Step 1: Open the app and navigate to user management. Step 2: Select Add User. Step 3: Choose code type - temporary, single-use, or recurring. Step 4: Set valid-from and valid-until date/time. For recurring, select day and time window. Step 5: Generate a PIN code (4-8 digits) - avoid 1234, 0000, or house number. Step 6: Name the code clearly (e.g. Airbnb Guest - Smith - June 2026). Step 7: Set max uses (single-use) or unlimited. Step 8: Enable notifications for temporary code usage. Step 9: Share the code via encrypted message (WhatsApp, Signal). Never post codes publicly. Step 10: Review active codes weekly and delete expired ones.

---

## Specifications and Comparison

| Code Type | Validity | Typical Use | Expiry Action |
| ---|---|---|--- |
| Time-limited | Date/time window | Airbnb guests, holiday let | Auto-expires at end time |
| Single-use | One operation only | Emergency access, delivery | Expires after first use |
| Recurring schedule | Weekly/daily pattern | Cleaner, dog walker | Active until deleted |
| Permanent | No expiry | Family members | Admin-only deletion |
| One-time use | Valid for 1 unlock | Parcel delivery | Expires after 5 mins unused |

## FAQ

<dl>

<dt>Can I set a temporary code for specific days and times?</dt>
<dd>Yes - most apps support recurring schedules. Set a code that works only on Tuesday mornings 9-11am for a cleaner.</dd>

<dt>Are temporary codes secure?</dt>
<dd>Yes - they are encrypted and stored locally. They cannot be used before activation or after expiry.</dd>

<dt>Can delivery drivers use temporary codes?</dt>
<dd>Some locks support one-time delivery codes through Amazon Key. For standard deliveries, use a single-use code. For more information see our <a href="../smart-locks/11-smart-lock-multi-user-access-guide.md">Smart Lock Multi-User Access Guide</a>.</dd>

</dl>
