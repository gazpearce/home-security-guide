## FAQ

<dl>

<dt>Can I unlock my door with voice commands?</dt>
<dd>Yes - but with security verification. Alexa requires PIN confirmation on Echo Show. Google requires Voice Match. Siri uses Face ID/Touch ID on iPhone.</dd>

<dt>Why does my lock take 5-10 seconds to respond to voice?</dt>
<dd>Voice commands are processed in the cloud and sent to the lock via WiFi. This 5-10 second delay is normal and within expected performance parameters.</dd>

<dt>Is voice control safe for door locks?</dt>
<dd>Yes - the security verification requirements make unauthorised voice unlock extremely difficult. Voice lock commands have no verification but can only lock, not unlock. For more information see our <a href="../smart-locks/22-smart-lock-for-sliding-patio-doors-guide.md">Smart Lock for Sliding Patio Doors Guide</a>.</dd>

</dl>
