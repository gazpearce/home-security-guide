# Smart Home Voice Control Setup Guide

Guide to voice control setup for UK smart homes covering Alexa, Google Assistant, and Siri voice commands, custom phrases, voice profiles, and multi-language support for diverse households.

---

## Voice Assistant Configuration

Three UK voice assistants: Alexa (45%), Google Assistant (30%), Siri (15%). Alexa setup: Alexa app > Devices > assign to room. Name devices clearly (Living Room Lamp, not Lamp 1). Enable skills for each brand. Google setup: Google Home > Add > assign to room. Use Matter for seamless setup. Apple Home setup: Home app > Add Accessory > scan HomeKit/Matter code. Voice profiles: Alexa recognises voices for personalised responses. Google Voice Match enables personalised voice unlock. Siri recognises different voices on HomePod.

---

## Useful Voice Commands and Custom Phrases

General: Alexa/Meta/Google/Siri, turn on [device name]. Room: turn off all lights in the kitchen. Set bedroom temperature to 18 degrees. Lock the front door. Routines: Alexa, good morning - opens blinds, starts kettle, plays radio, announces weather. Alexa, Im leaving - locks doors, arms alarm, turns off lights. Alexa, goodnight - same plus sets thermostat to 16C. Custom phrases: create Alexa Routines with custom voice triggers like Alexa, movie time (dim lights, close blinds, turn on TV). Alexa, guests are here (disarm alarm, unlock door, set 21C). Multi-language households: Alexa supports English and select other languages. Google supports multi-language mode - switch between languages without changing settings. Accessibility: all platforms support voice control for users with mobility or dexterity issues. Enable voice control for all smart home functions. Alexa Show and Tell for visually impaired users (identifies objects via Echo Show camera).

## Specifications and Comparison

| Feature | Alexa | Google Assistant | Siri (HomeKit) |
| ---|---|---|--- |
| Voice unlock | PIN confirmation | Voice Match | Home Key (NFC) |
| Multi-language | Limited | Good multi-language mode | System language only |
| Routines | Alexa Routines | Google Routines | Scenes + Shortcuts |
| Household profiles | Alexa profiles | Voice Match | Family Sharing |
| Accessibility | Show and Tell | Voice Access | VoiceOver + Siri |

## FAQ

<dl>

<dt>Can I use voice to unlock my door?</dt>
<dd>Alexa requires PIN confirmation on Echo Show. Google supports voice unlock with Voice Match. Siri can unlock via Home Key on iPhone.</dd>

<dt>Does voice control work without internet?</dt>
<dd>No - voice processing requires cloud. When internet is down, use app or keypad as backup.</dd>

<dt>Can I customise voice commands?</dt>
<dd>Yes - Alexa allows custom routine triggers. Create phrases like Alexa, movie time or Alexa, Im off. For more information see our <a href="../smart-home/12-smart-home-energy-monitoring-guide.md">Smart Home Energy Monitoring Guide</a>.</dd>

</dl>
