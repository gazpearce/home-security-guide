# Smart Home WiFi Network Requirements Guide

Guide to WiFi network requirements for UK smart homes covering mesh vs single router, 2.4GHz vs 5GHz band management, device limits, and optimising WiFi for 20+ smart devices.

---

## WiFi Network Planning for UK Smart Homes

A typical UK smart home has 10-20 devices. Standard ISP router handles 15-25 but degrades beyond. Mesh WiFi recommended for 15+ devices. Popular UK mesh: TP-Link Deco XE75 (£200-£300 for 3-pack, WiFi 6E), Google Nest WiFi Pro (£120-£200 per node), Eero 6+ (£150-£250 for 3-pack), BT Whole Home WiFi (£130-£200 for 3-pack), Asus ZenWiFi AX (£250-£400 for 2-pack).

---

## WiFi Configuration for Smart Home Devices

Band management: most smart devices only support 2.4GHz. Enable separate SSIDs or temporarily disable 5GHz when pairing. Use 2.4GHz for smart home hub for max range through brick walls. Channel selection: 1, 6, or 11 are non-overlapping in UK. Use WiFi analyser to find least congested. In UK terraced houses, channel 1 or 11 is often best. QoS: Prioritise security cameras and streaming. Device naming: name clearly in router. Security: change default admin password, enable WPA3 or WPA2, create guest network. Check router firmware monthly.

## FAQ

<dl>

<dt>Do I need mesh WiFi for a smart home?</dt>
<dd>For under 15 devices, ISP router suffices. For 15-30+ devices or multiple floors, mesh significantly improves reliability.</dd>

<dt>Why wont my smart device connect?</dt>
<dd>Most only support 2.4GHz. Temporarily disable 5GHz or use separate 2.4GHz SSID when pairing.</dd>

<dt>How many devices can my WiFi handle?</dt>
<dd>ISP routers: 15-25. Mesh: 50-100+. For 25+ smart devices, mesh is recommended. For more information see our <a href="../smart-home/08-smart-speaker-and-display-setup-guide.md">Smart Speaker and Display Setup Guide</a>.</dd>

</dl>
