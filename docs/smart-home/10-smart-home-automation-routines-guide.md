# Smart Home Automation Routines Guide

Guide to creating smart home automation routines for UK homes covering Alexa Routines, Google Routines, Apple Shortcuts, Home Assistant automations, and practical UK lifestyle templates.

---

## Automation Routine Platforms

Alexa Routines: triggers include voice, schedule, location, sensor. Actions include smart home, music, announcements, wait up to 4 hours, conditions. Google Routines: triggers include voice, schedule, sunrise/sunset, home/away. Apple Home/Shortcuts: triggers include time, location, sensor, device. Home Assistant: unlimited triggers/actions, scriptable logic, 2000+ integrations. For most UK homes, Alexa Routines offer the best balance. Power users should consider Home Assistant on Raspberry Pi 5 (£100-£150).

---

## Practical UK Automation Templates

Good Morning (trigger: sunrise): open blinds, set thermostat to 21C, turn on kettle via smart plug, play BBC Radio 4, announce weather. Good Night (trigger: Alexa goodnight): lock doors, turn off all lights, set thermostat to 16C, arm alarm, turn off TVs via smart plugs, close blinds. Arrive Home (geofencing): disarm alarm, unlock door, turn on hall/living room lights, set 21C, announce deliveries. Leave Home (geofencing): lock doors, arm alarm, turn off lights, set 16C, turn off entertainment, close blinds. Guest Mode: set 20C, disarm, unlock, play welcome. Energy Saving (no motion 30 min): turn off lights, eco 14C. Fire Safety (smoke detected): unlock all doors, turn on all lights, turn off boiler, announce evacuation.

## Specifications and Comparison

| Platform | Triggers | Actions | Ease of Use | Best For |
| ---|---|---|---|--- |
| Alexa Routines | Voice, schedule, location, sensor | Smart home, music, announcements, wait | Easy | Most UK households |
| Google Routines | Voice, schedule, location | Smart home, music, news, info | Easy | Google ecosystem users |
| Apple Home/Shortcuts | Time, location, sensor, device | Smart home, apps, settings | Medium | Apple households |
| Home Assistant | Every sensor, time, state, event | Smart home, script, API, 2000+ | Complex | Enthusiasts, max control |

## FAQ

<dl>

<dt>What is the most useful UK routine?</dt>
<dd>Good Night - locks doors, turns off lights, sets 16C, arms alarm. Covers security, energy, comfort in one command.</dd>

<dt>Can routines run without voice?</dt>
<dd>Yes - schedule, location (geofencing), or sensor triggers. Mix of triggers works best.</dd>

<dt>Can I create routines across device brands?</dt>
<dd>Yes - Alexa/Google support different brands. One routine can control Hue lights, Nest thermostat, Ring alarm, Yale lock. For more information see our <a href="../smart-home/09-smart-home-wifi-network-requirements-guide.md">Smart Home WiFi Network Requirements Guide</a>.</dd>
