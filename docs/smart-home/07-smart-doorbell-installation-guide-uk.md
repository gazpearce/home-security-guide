# Smart Doorbell Installation Guide UK

Complete guide to installing a smart video doorbell for UK homes covering wired vs battery models, Ring, Nest, Eufy, and Hive doorbell installation, transformer checks, and WiFi considerations.

---

## Choosing a Smart Doorbell for UK Homes

Three types: wired (connected to existing 8-24V AC wiring, Ring Pro 2 £180-£250, Nest Wired £180-£220), battery (easy install, Ring Battery Plus £100-£150, Eufy 2K Battery £80-£130), wireless/digital (Hive Video Doorbell £120-£160). Key UK considerations: IP54 minimum weather resistance, colour night vision for dark UK winters, wedge mounts for brick reveals, local storage option (Eufy). All need 2.4GHz WiFi within 10m of front door.

---

## Installation Steps and Configuration

Battery: Charge fully (2-4 hours), download app, pair to 2.4GHz WiFi, mount at 1.2-1.4m with Fischer UX plugs for brick. Wired: Check transformer (8-24V AC, 10VA+), turn off power, remove existing doorbell, connect wires, install chime bypass module (Ring Pro 2), mount at 1.2-1.4m. Configuration: set motion zones to focus on doorstep (not pavement), enable package detection, adjust video (1080p for UK upload speeds), connect to Alexa/Google. For terraced houses, adjust motion zones to avoid neighbours property.

## FAQ

<dl>

<dt>Can I install on a uPVC door frame?</dt>
<dd>Yes - adhesive mounts or short screws work. For brick reveals, use screw fixings with wall plugs.</dd>

<dt>Do I need a subscription?</dt>
<dd>Ring and Nest require subscriptions (£3-£6/month) for recording. Eufy stores locally with no subscription.</dd>

<dt>What if someone steals my doorbell?</dt>
<dd>Most have anti-theft screws and tamper alerts. Report serial number as stolen to manufacturer. For more information see our <a href="../smart-home/06-smart-blinds-and-curtains-guide.md">Smart Blinds and Curtains Guide</a>.</dd>

</dl>
