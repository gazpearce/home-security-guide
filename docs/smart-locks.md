# Smart Locks

Smart locks offer keyless entry and remote access control for modern homes.

## Benefits

- No physical keys needed
- Temporary access codes for guests
- Remote locking/unlocking via app
- Integration with smart home systems
- Activity logs showing who entered when

## Types

- **Keypad Locks** — PIN code entry
- **Biometric Locks** — Fingerprint scanning
- **Bluetooth Locks** — Auto-unlock via phone proximity
- **Wi-Fi Locks** — Full remote control anywhere

## Security Considerations

Look for British Standard TS 007 3-star rating and Sold Secure accreditation.
