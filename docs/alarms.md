# Alarm Systems

Burglar alarms are an effective deterrent and can reduce home insurance premiums.

## Types

- **Wired Alarms** — Reliable, no battery concerns, professional install
- **Wireless Alarms** — DIY-friendly, flexible placement
- **Smart Alarms** — App-controlled, integrates with other smart home devices

## Key Components

- Control panel
- Door/window sensors
- Motion detectors (PIR)
- Glass break detectors
- Siren (internal and external)
- Keypad or fob

## Monitoring Options

- **Self-monitoring** — Free, app alerts
- **Keyholder monitoring** — Neighbour or friend notified
- **ARC monitoring** — Professional Alarm Receiving Centre
