# Security Tips

## Basic Measures

- Always lock doors and windows, even when home
- Install a visible alarm box
- Use timer switches for lights when away
- Keep hedges trimmed to avoid hiding spots
- Join or start a Neighbourhood Watch scheme

## Advanced Security

- Install outdoor security lighting with PIR sensors
- Use video doorbells for package security
- Consider a safe for valuables
- Secure garden sheds and outbuildings
- Review your home security annually

## Insurance

Many UK insurers offer discounts for fitted alarms and CCTV. Check with your provider.
